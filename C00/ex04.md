## ft_is_negative.c

```c++
#include <unistd.h>

void ft_is_negative(int n){
  char out;
  
  if(n >= 0){
    out = 'P';
  }
  else {
    out = 'N';
  }
  write(1, &out, sizeof(out));
}
```
